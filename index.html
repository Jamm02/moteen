Projects



    Going Beyond Backscatter


  • - An ultra low power trasmitter based on free running tunnel diode oscillators that aims to surpass the drawbacks of backscatter technology.
  • - Developed a 2-MFSK baseband generator using FreeRTOS utilizing just 800 microwatts in the process.
  • - Research and development to mitigate unstable nature of Tunnel Diodes with RF isolators, Phase locked loops, EMI/RFI shielding.
  • - The work was submitted in MobiCom 2023 as a poster presentation which won the undergraduate category of the Student Research Competition. The article can be found here.




    Mainline Development of Allwinner D1


  • - The aim of the project was to provide support for the D1 SoC on U-Boot
  • - Patched configuration files, device tree files for the new SoC.
  • - Patched USB, MMC, GPIO and emac drivers so as to make them generic and work across RISC-V and ARM architecture.
  • The link for the codebase can be found here.





    Upside Down Labs


  • - Upside Down Labs offers affordable open-source hardware and software solutions focused on developing systems for neuroscience and human-computer interfaces.
  • - Developed a Low cost production grade ECG monitor using Biopotential sensor. Link to firmware.
  • - Developed firmware for heartrate detection and routed the data to smartphone through BLE. Link to firmware
  • - Developed a prototype for a bionic arm. The demo can be found here.




    ESP32-audio-router


  • - The project aimed to route audio from any source to a micro-controller using bluetooth.
  • - Deployed I2S protocol for trasmitting audio packets over bluetooth.
  • - Developed drivers for establishing communication between external DAC and microcontroller using serial and I2C comunication protocol.
  • - Learning outcomes: Bluetooth stack, audio profiles and protocols, I2C/SPI drivers.
  • - Project repository.
